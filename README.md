# OPTIMIZACIÓN SEO: CONVERTIR A WEBP
Creado originalmente el 29/09/2025
SEO Core Web Vitals check: convertir cualquier formato de imagen al formato .webp que es el ideal para el posicionamiento SEO estático en una web.

## Requisitos
- Necesario: crear capreta de origen y destino y representarlas en el código en la parte:
carpeta_origen = r"D:/"
carpeta_destino = r"D:/"
- Instalar librería Pillow.
